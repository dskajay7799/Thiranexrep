About ShopEasy

What is ShopEasy?

ShopEasy is an e-commerce product catalog built as a final year capstone project for our Web Development course. The project uses HTML, CSS, JavaScript on the frontend and Python Flask on the backend.

The goal was to build a full-stack web application from scratch that connects a frontend to a backend API. We learned a lot about how real shopping websites work during this project.

Technologies Used

  • HTML and CSS for the frontend layout and design
  • Vanilla JavaScript with fetch() and async/await for API calls
  • Python Flask for the backend REST API
  • JSON file for storing product data
  • Flask-CORS to handle cross origin requests
  • Gunicorn for production deployment on Render

Features

  • Browse all products from the catalog
  • Search products by name, category or description
  • Filter by category and sort by price or rating
  • View detailed product information
  • Responsive layout for mobile and desktop
  • RESTful API with basic error handling
